## vscode

### インストール手順（red hat）
1. インストーラをダウンロード  
[https://code.visualstudio.com/download](https://code.visualstudio.com/download)
1. ターミナルからyumでインストール
```
$ yum install code-1.32.3-1552607091.el7.x86_64.rpm
```
