## git

### 環境
- ワークツリー  
自身の作業環境
- ステージ  
ワークツリーでの変更内容をコミットするための準備環境
- (リモート)リポジトリ  
変更内容の保存

ワークツリーにて作業した内容は一度ステージに上げないとリポジトリに上げることができない。

---

### 設定
gitの設定は以下コマンドで行う
```
$ git config
```

```
--global        設定がPC全体に適用される
```

ex. エイリアス  
```git status```を```git st```に
```
$ git config --global alias.st status
```

---

### gitignore
gitで管理しないファイルを指定する
```
# #はコメント

# 指定したファイルを除外
sample.html
# 対象のディレクトリ直下をすべて除外
test/
# パターンマッチング
test/*.html
```
